OBJETOS: LOS OBJETOS TIENEN MUCHA INFORMACIÓN SOBRE UNA SOLA COSA. SE INSTANCIAN USANDO LLAVES. 
LOS OBJETOS USAN UN CONCEPTO LLAMADO PARES DE CLAVE:VALOR. LA CLAVE (KEY) ES EL IDENTIFICADOR Y EL VALOR (VALUE) ES EL VALOR QUE QUEREMOS GUARDAR EN ESA CLAVE.
LA SINTAXIS ES CLAVE: VALOR. LOS VALORES PUEDEN SER CUALQUIER TIPO DE DATO DE JS; NUMERO CADENA BOOLEANO FUNCIÓN U OTRO OBJETO.

PROPIEDADES: CON LA NOTACIÓN DE PUNTOS PODEMOS LLAMAR AL NOMBRE DEL OBJETO, UN PUNTO Y EL NOMBRE DE LA CLAVE. POR EJEMPLO LA PROPIEDAD .LENGTH ES UNA PAR CLAVE-VALOR.
PARA ELIMINAR UNA PROPIEDAD SE USA LA PALABRA CLAVE DELETE Y EL OBJETO, UN PUNTO Y LA PROPIEDAD QUE SE DESEA ELIMINAR.

MÉTODOS: DENTRO DE LOS OBJETOS, LOS VALORES SE PUEDEN ESTABLECER EN FUNCIONES. LAS FUNCIONES GUARDADAS DENTRO DE UN OBJETO SE DENOMINAN MÉTODOS. SE PUEDE ESTABLECER UNA CLAVE PARA UN NOMBRE Y QUE EL VALOR SEA UNA FUNCIÓN.
LOS MÉTODOS SERÁN LLAMADOS USANDO LA NOTACIÓN DE PUNTOS Y PARÉNTESOS FINALES.

BUCLES FOR..IN: JS TIENE UN SEGUNDO TIPO DE BUCLE INTEGRADO LLAMADO "FOR... IN LOOP". EL BUCLE ESTÁNDAR NO FUNCIONARA PARA EL OBJETO PORQUE ESTE NO TIENE ÍNDICES NUMÉRICOS COMO LAS MATRICES O ARREGLOS.
LA SINTAXIS ES SIMILAR, COMIENZA IGUAL PERO DENTRO DEL PARENTESIS TIENE UNA PALABRA CLAVE "IN" Y EL NOMBRE DEL OBJETO. ESTO RECORRERA CADA CLAVE DEL OBJETO Y FINALIZARÁ CUANDO SE HAYAN ITERADO TODAS LAS CLAVES. PODEMOS USAR ESTA CLAVE Y LA NOTACIÓN DE CORCHETES PARA ACCEDER AL VALOR ASOCIADO CON ESA CLAVE.

NOTACIÓN DE PUNTOS VS NOTACIÓN DE CORCHETES:

CON LA NOTACIÓN DE PUNTOS SE PUEDE LLAMAR AL NOMBRE DEL OBJETO, UN PUNTO Y EK NOMBRE DE LA CLAVE; LA NOTACIÓN DE CORCHETES ES COMO LLAMAR A UN ELEMENTO EN UN ARREGLO O MATRIZ, AUNQUE CON CORCHETES DEBEMOS UTILIZAR UNACADENA O NÚMERO, O UNA VARIABLE QUE APUNTE A UNA CADENA O NÚMERO. SE LLAMA A CADA CLAVE ENVOLVIENDOLA ENTRE COMILLAS.
