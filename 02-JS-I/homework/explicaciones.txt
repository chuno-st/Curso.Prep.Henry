VARIABLES: UNA VARIABLES ES UNA FORMA DE ALMACENAR UN VALOR PARA USAR MÁS TARDE;
ES COMO CREAR UNA CAJA Y PONER ALGO DENTRO DE ELLA, ALGO QUE LUEGO USAREMOS EN ALGÚN MOMENTO;

STRINGS: LOS STRINGS SON BLOQUES DE TEXTO, CARACTERES UNIDOS, HILADOS, ENCADENADOS;
PUEDEN CONTENER NÚMEROS Y SIEMPRE SE DEFINIRÁN ENTRE COMILLAS (SIMPLES O DOBLES COMILLAS);

FUNCIONES: LAS FUNCIONES SON TIPOS PARTICULARES DE OBJETOS, OBJETOS INVOCABLES;
A TRAVÉS DE ELLAS PODEMOS CALCULAR, CAMBIAR Y REALIZAR OPERACIONES CON VARIABLES;
ARGUMENTOS: LOS ARGUMENTOS SON LAS VARIABLES NECESARIAS (PUEDE SER UNA O MÁS DE UNA) 
QUE NECESITA LA FUNCIÓN PARA EJECUTAR EL CÓDIGO QUE CONFIGURAMOS; SE SEPARAN POR COMAS;
RETURN: ES LO QUE QUEREMOS DEVOLVER DE UNA FUNCIÓN; LA FORMA EN QUE LOS DATOS SALEN DE UNA FUNCIÓN;

DECLARACIONES IF: LA PALABRA CLAVE IF EN NUESTRA FUNCIÓN VERIFICA SI ALGO ES VERDADERO
EJECUTANDO EL CÓDIGO QUE CONFIGURAMOS O DE LO CONTRARIO LO OMITE DEVOLVIENDO QUE ES FALSO;

VALORES BOOLEANOS: EN JAVASCRIPT SE USAN LOS VALORES BOOLEANOS PARA SIGNIFICAR VERDADERO O FALSO;
LOS VALORES POSIBLES DE UN DATO BOOLEANO EN JS SON TRUE O FALSE;
